# 🔐 Secure Password Generator

A modern, privacy-focused password generator built with pure HTML, CSS and JavaScript.

Generate strong passwords instantly in your browser without sending any data to external servers.

---

## ✨ Features

* 🔑 Generate secure random passwords
* 📏 Custom password length (6–64 characters)
* 🔠 Uppercase letters support
* 🔡 Lowercase letters support
* 🔢 Numbers support
* ✳️ Special symbols support
* 📋 One-click copy to clipboard
* 📊 Password strength indicator
* 🎨 Glassmorphism UI design
* 🌙 GitHub-inspired dark theme
* 📱 Responsive design
* ⚡ Fast and lightweight
* 🔒 100% client-side generation

## 🔒 Privacy

All passwords are generated locally in your browser.

This application:

* Does not collect user data
* Does not store passwords
* Does not use cookies
* Does not send passwords to any server

Your generated passwords never leave your device.

---

## 📈 Password Strength Rules

The strength indicator evaluates passwords based on:

* Length
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

Levels:

| Score  | Rating |
| ------ | ------ |
| Low    | Weak   |
| Medium | Medium |
| High   | Strong |
